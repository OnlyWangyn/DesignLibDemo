# DesignLibDemo
a simple demo for Android Design Support Library,it includes
1 Navigation View
2 Floating Action Button
3 Snackbar
4 CoordinatorLayout and floating action buttons
5 Collapsing Toolbars
above all only work on buildToolsVersion 22
